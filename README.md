#           Проектная работа:
#           Mesto Russia.

###  В проект включен профиль пользователя с краткой информацией  
### о себе (имя и чем занимается). Пользователь может просматривать  
### понравившиеся картинки и лайкать их. На станице реализовано  
### всплывающее окно на основе input, button, form. И используется  
### логика для popup на js. Еще используются выравнивание grid и  
### flex.   
###  Страница адаптирована под устройства с шириной экрана от 320px  
### до 1280px.
###         [Ссылка на страницу:](https://mihhaa.github.io/mesto/)
